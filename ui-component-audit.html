Logo / festival title — not a centralized component

Both the app wordmark and the festival-page title are the same raw <Link> in src/components/layout/Header.tsx:60-66 — one element whose text and target swap depending on route, styled inline with font-mono font-bold text-accent text-base tracking-tight truncate. It is not built on Heading (closest visual match: content variant), and there is no dedicated Logo or FestivalTitle component.
